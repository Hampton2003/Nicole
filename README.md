# 💕 For Nicole - A Romantic Website

A beautiful, interactive romantic website created with love for Nicole.

## ✨ Features

- **Floating Hearts Animation** - Romantic hearts floating in the background
- **Smooth Scroll Navigation** - Elegant navigation with active state tracking
- **Interactive Photo Gallery** - Click photos to view them in a lightbox
- **Animated Love Letter** - Click the envelope to reveal a heartfelt letter
- **Responsive Design** - Looks great on all devices
- **Parallax Effects** - Subtle mouse movement effects
- **Scroll Reveal Animations** - Elements animate as you scroll
- **Mobile Menu** - Hamburger menu for mobile devices

## 🚀 How to Use

1. **Download all files** from this folder
2. **Upload to GitHub**:
   - Create a new repository
   - Upload all files maintaining the folder structure
   - Enable GitHub Pages in repository settings
   - Your site will be live at `https://yourusername.github.io/repository-name`

3. **Or open locally**:
   - Simply open `index.html` in your browser

## 🎵 Adding Music

To add background music:
1. Add your romantic song file to the project folder
2. In `js/main.js`, find the `initMusicToggle` function
3. Uncomment the audio source line and add your song path:
   ```javascript
   audio.src = 'your-song.mp3';
   ```

## 💝 Customization

- Edit the letter text in `index.html` in the `#letter` section
- Change colors by modifying CSS variables in `css/style.css`
- Add more photos to the gallery by duplicating gallery items
- Customize the "Why I Like Her" reasons in the `#reasons` section
